<h1 align="center">mise-gib</h1>

<p align="center">
  <a href="https://mise.jdx.dev">Mise</a> plugin for <a href="https://github.com/jbadeau/gib">Gib</a>.
</p>

---

## Install

```sh
mise plugin install gib https://github.com/jbadeau/mise-gib.git
```

```sh
mise install gib@latest
```

```sh
mise use gib@latest
```

## Supported Platforms

| OS | Architecture |
|---|---|
| Linux | amd64, arm64 |
| macOS | amd64, arm64 |
| Windows | amd64, arm64 |
